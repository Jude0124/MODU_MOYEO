## 2. 프로젝트 소개 및 개요

국비교육 블라인드 커뮤니티 웹 프로젝트는 주기능으로 인증기능을 도입하여 인증된 기수강생이 교육에 대해 후기, 별점을 작성하여 예비수강생에게 과목, 학원 평가에 대한 정보를 제공해주고 부기능으로 각 학원별 커뮤니티, 채팅등을 통한 수강생간 소통, 학원 비교, 성향테스트 등을 제공하고 있습니다.
